# navigator-drone
Drone Navigator 🚁

О проекте

Drone Navigator — это прототип наземной станции управления дроном, разработанный на JavaFX с использованием Яндекс Карт.

Основная идея проекта — предоставить оператору возможность задавать точку назначения на карте и визуально наблюдать движение дрона к выбранной цели.

Проект создавался как первый шаг к полноценной системе управления беспилотниками с дальнейшей интеграцией PX4 и MAVSDK.

---

Возможности

- Отображение карты через Яндекс Maps API
- Выбор точки назначения кликом мыши
- Отображение дрона на карте
- Плавное движение дрона к заданной точке
- Передача координат между Java и JavaScript
- Телеметрия (координаты, статус, батарея)
- Подготовка к интеграции с реальным автопилотом

Архитектура

Проект состоит из нескольких основных компонентов:

MainWindow

Главное окно приложения.

Отвечает за:

- запуск JavaFX приложения;
- отображение карты;
- обработку пользовательских действий;
- отображение телеметрии.

MapBridge

Мост между Java и JavaScript.

Позволяет:

- получать координаты клика по карте;
- передавать данные из Java в браузер.

MockDroneService

Сервис симуляции дрона.

Отвечает за:

- движение виртуального дрона;
- обновление координат;
- симуляцию батареи.

map.html

Страница с Яндекс Картой.

Реализует:

- отображение карты;
- маркер дрона;
- маркер цели;
- взаимодействие с Java приложением.
